## Installation

### Prerequisites

- Python >= 3.10
- [pipx](https://pipx.pypa.io/stable/installation/#on-linux)

```sh
pipx install setch-py
```

## Usage

```sh
setch
```

## Uninstall

```sh
pipx uninstall setch-py
```
