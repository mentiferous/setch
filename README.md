# setch

A utility for retrieving system information on Linux

## Installation

### Prerequisites

- [git](https://git-scm.com/downloads/linux)
- [Python](https://www.python.org/downloads/source)

```sh
git clone https://github.com/mentiferous/setch.git
```

```sh
cd setch/src
```

## Usage

```sh
python -m setch
```
